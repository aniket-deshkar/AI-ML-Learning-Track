# AI & Machine Learning Learning Track

Welcome to my AI and Machine Learning learning track repository! This collection contains my experimental exercises, code snippets, and learning materials acquired over various studies, covering foundational and advanced topics in Artificial Intelligence, Machine Learning, and Software Architecture.

## Repository Structure

- **`FastAPI`**: Exercises and basics on building robust backend APIs with FastAPI, routing, and authentication.
- **`LLMs`**: Foundational concepts and examples working with Large Language Models (LLMs), prompt engineering, LangChain, and LlamaIndex.
- **`PythonTrack`**: Core Python programming exercises covering OOPs, decorators, file handling, asynchronous IO, and more.
- **`machine_learning`**: Jupyter notebooks and scripts covering machine learning basics such as regression techniques.
- **`nlp`**: Natural Language Processing exercises.
- **`langgraph`**: Experiments and exercises related to stateful AI agents using LangGraph.
- **`titanic`**: Exploratory Data Analysis (EDA) and straightforward classification tasks on the classic Titanic dataset.

## Setup & Running

It is recommended to use an isolated Python virtual environment for each subdirectory if external dependencies are required. Many of the exercises contain their own `requirements.txt` or `.ipynb` notebooks.

```bash
python -m venv .venv
source .venv/bin/activate  # Or `.venv\Scripts\activate` on Windows
pip install [required-libraries]
```
