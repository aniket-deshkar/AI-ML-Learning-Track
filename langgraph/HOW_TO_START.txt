How To Start (Short)

1) Open terminal in project root:
   cd /home/{user_DIR}/PycharmProjects/Path2AI/langgraph

2) Activate environment:
   source .venv/bin/activate

3) Ensure .env has:
   LANGSMITH_API_KEY=...
   LANGSMITH_TRACING=true
   (At least one model key)
   GEMINI_API_KEY=... or GROQ_API_KEY=... or HF_TOKEN=...

4) Start API server:
   langgraph dev --config langgraph.json --studio-url https://smith.langchain.com --port 2024

5) Open Studio URL printed in terminal.
   In Graph mode -> View Raw -> paste input JSON -> Submit.
